## 🎯 Key Features

- **Complete Graph Support**: All combinations of weighted/unweighted × directed/undirected
- **Rich Algorithm Library**: 15+ implemented algorithms
- **Fluent API**: Intuitive, chainable method calls
- **High Performance**: Optimized for single-machine memory usage
- **Comprehensive Testing**: Extensive unit tests and benchmarks
- **Type Safety**: Full generic type support
